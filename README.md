# My-journey

## Affichage des projets dans la console

Le script Python `main.py` affiche les projets dans la console avec des couleurs pour une meilleure lisibilité. Voici comment l'utiliser :

1. Assurez-vous que vous avez Python 3 et le module `termcolor` installé.
2. Ouvrez un terminal ou une console dans le répertoire contenant `main.py`.
3. Exécutez la commande `python main.py`.
4. Les projets seront affichés avec des couleurs dans la console.
